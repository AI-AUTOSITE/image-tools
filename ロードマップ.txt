## Week 3: ユーザー管理システム

### 3段階ユーザーシステムの実装
```typescript
// Supabase Schema
CREATE TABLE users (
  id UUID PRIMARY KEY,
  email TEXT UNIQUE,
  tier TEXT DEFAULT 'free', -- guest/free/pro
  created_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE usage_logs (
  id UUID PRIMARY KEY,
  user_id UUID REFERENCES users(id),
  tool_id TEXT NOT NULL,
  action TEXT NOT NULL,
  created_at TIMESTAMP DEFAULT NOW()
);

// 日次使用量リセット
CREATE OR REPLACE FUNCTION reset_daily_usage()
RETURNS void AS $$
BEGIN
  -- 毎日0時にリセット
  UPDATE users SET daily_usage = 0;
END;
$$ LANGUAGE plpgsql;
```

### 使用制限の実装
```typescript
const TIER_LIMITS = {
  guest: {
    daily: 3,      // ゲスト：3回/日

  },
  free: {
    daily: 10,      // 無料：10回/日

    daily: Infinity, // プロ：無制限
```
